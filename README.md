# skill-note
## 使用技術
Laravel+Docker
## 簡単な説明
 気になった技術をまとめるだけのノート

## インストール

```
$ git clone https://github.com/t1k2a/skill-note.git
$ cd skill-note
$ docker-compose up -d --build
$ docker-compose exec app ash
$ composer install
$ cp .env.example .env
$ php artisan key:generate
$ php artisan migrate
```
